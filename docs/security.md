# Security

Neksus JobSpec security policy is maintained in the repository root:

- `SECURITY.md` (root file in the source repository)
- GitHub view: <https://github.com/NeksusAI/NeksusJobSpec/blob/main/SECURITY.md>

Key points:

- Report vulnerabilities privately to `security@neksusai.com`.
- Latest supported line is `0.2.x`.
- Public issues should not be used for unpatched security findings.
