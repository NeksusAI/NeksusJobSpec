# neksus_jobspec_cli.commands

CLI command modules grouped by workflow domain.

## Files

- `spec.py`: single-spec workflows.
- `render.py`: batch project render.
- `feed.py`: multi-spec feeds/sitemap.
- `check.py`: project checks.
- `config.py`: project config read/update.
- `themes.py`: theme metadata commands.
- `init.py`: project initialization.
- `version.py`: version output.
- `common.py`: shared output/error helpers.

## Rule

Use this layer for argument parsing and output formatting only.
